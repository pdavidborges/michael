Este projeto foi desenvolvido como meu primeiro projeto Front-End, com o objetivo de praticar os fundamentos de HTML5 e CSS3 na criação de uma página moderna, responsiva e visualmente organizada.

A aplicação consiste em uma landing page inspirada no filme “Michael”, apresentando informações como título, gênero, sinopse e botão para assistir ao trailer. Durante o desenvolvimento, foram aplicados conceitos importantes de estruturação semântica, estilização visual e responsividade para diferentes tamanhos de tela.

Entre os principais recursos utilizados no projeto estão:

Estrutura semântica com HTML5
Estilização completa utilizando CSS3
Layout com Flexbox
Responsividade com Media Queries
Background personalizado com imagem
Botão interativo com efeito hover
Versão otimizada para impressão utilizando @media print
Organização visual com foco em experiência do usuário

Além da parte visual, o projeto também serviu como prática para organização de código, separação entre estrutura e estilo, além da compreensão de boas práticas iniciais no desenvolvimento web.
